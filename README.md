# Annual Calendar (Horizontal Year View)

This is a simple annual calendar website:
- 12 months as rows
- Days 1–31 as columns
- Weekend shading
- Today marker
- Event bars from pasted ICS calendar links

## Deploy (Vercel)
1. Push this repo to GitHub
2. In Vercel: New Project → Import Git Repository → select this repo → Deploy

## Local run (optional)
```bash
npm install
npm run dev
```
Open http://localhost:3000
